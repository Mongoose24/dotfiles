1. Clone dotfiles repo
```
git clone https://github.com/Mongoose24/dotfiles.git ~/dotfiles 
```

2. cd into dotfiles
```
cd ~/dotfiles
```

3. chmod script(s)
```
chmod +x deb-in.sh
```
or
```
chmod +x arch-in.sh
```
4. check script contents if you want, then run it
```
./deb-in.sh
```
or
```
./arch-in.sh
```
